Template to start a new Typescript project

# Install

```bash
git clone git@github.com:Skycatch/ts-project-template.git my-project-name
cd my-project-name
npm install
npm run dev
```

# Tests

```bash
npm run test
npm run test:watch
```

# Build

```bash
npm run build
npm start
```

# Features

- Path alias
- Tests
- Env vars
- Format
- Tsconfig
- Github actions
